# 🕷️ 보니의 웹 크롤링 프로그램

네이버 카페, 블로그, 유튜브를 동시에 검색하여 최신 정보를 수집하는 종합 크롤링 프로그램입니다.

## ✨ 주요 기능

### 🔍 통합 검색
- **네이버 카페, 블로그, 유튜브** 동시 검색
- **최신순 정렬**로 가장 최근 정보 우선 제공
- **1개월 이내** 게시물 필터링

### 🎯 고급 검색
- **AND 조건 검색**: 여러 키워드 조합 가능 (키워드1 AND 키워드2)
- **제외 키워드**: 원하지 않는 내용 실시간 필터링
- **동적 키워드 추가**: "+" 버튼으로 키워드 계속 추가 가능

### 💾 데이터 관리
- **Excel 내보내기**: 검색 결과를 Excel 파일로 저장
- **Excel 불러오기**: 기존 Excel 파일 로드 및 결과 누적
- **즐겨찾기**: ⭐ 표시로 중요한 결과 관리
- **검색 기록**: 이전 검색어 자동 저장 및 관리

### 🎨 사용자 인터페이스
- **직관적인 GUI**: tkinter 기반의 깔끔한 인터페이스
- **실시간 진행률**: 검색 진행 상황 실시간 표시
- **한글 완전 지원**: 모든 UI가 한국어로 제공

## 📁 파일 구조

- `search_history.json` - 검색 기록 저장 파일

### 🚀 실행 파일 (데스크탑)
- `보니의 웹 크롤링 프로그램 실행.command` - 터미널 실행 (추천)
- `보니의웹크롤링프로그램.app` - 앱 형태 실행 (예쁜 아이콘)

### 📊 예시 결과 파일
- `20250617-서울 전시 AND 무료.xlsx` - 검색 결과 예시

## 🛠️ 설치 및 실행

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행
```bash
# 방법 1: Python으로 직접 실행
python3 naver_search_complete.py

# 방법 2: 데스크탑 아이콘 더블클릭 (추천)
# "보니의 웹 크롤링 프로그램 실행.command" 더블클릭
```

## 📝 사용법

1. **키워드 입력**: 검색하고 싶은 키워드 입력
2. **키워드 추가**: "+" 버튼으로 AND 조건 키워드 추가
3. **제외 키워드**: 제외하고 싶은 키워드 입력
4. **검색 실행**: "🔍 검색 시작" 버튼 클릭
5. **결과 확인**: 카페, 블로그, 유튜브 결과를 탭별로 확인
6. **즐겨찾기**: 중요한 결과에 ⭐ 표시
7. **Excel 저장**: "📊 Excel로 내보내기" 버튼으로 결과 저장

## 🎯 검색 팁

- **구체적인 키워드 사용**: "이촌동 아파트" vs "아파트"
- **AND 조건 활용**: "서울 AND 전시 AND 무료"
- **제외 키워드 활용**: 광고성 게시물 필터링
- **정기적인 검색**: 최신 정보 업데이트 확인
